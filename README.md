Personal Portfolio Website

📌 Overview

This is my personal portfolio website developed using HTML5 and CSS3. It showcases my education, technical skills, projects, certifications, resume, and contact information in a responsive and interactive design.

✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Sticky navigation bar with smooth scrolling
- Hero section with animations
- About Me section
- Education details in a table
- Skills section with hover effects
- Projects showcase
- Certifications section
- Social media links
- Resume upload form
- Contact section with Email, Phone, and LinkedIn
- Audio and video introduction
- Embedded Google Maps location
- Modern UI with gradients, animations, and glassmorphism

🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)

📂 Sections

- Home
- About
- Education
- Skills
- Projects
- Certifications
- Social Links
- Contact Information
- Resume
- Contact Me

🚀 How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click index.html or open it using any modern web browser.
